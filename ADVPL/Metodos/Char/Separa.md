# Resumo:
A função Separa() funciona como a função Split()



Exemplo: 

local cTexto  := "Gabriel, Fabio, Rodrigo, Isabella, Sandra"

aSeparado := Separa(cTexto, ",")


O Resultado dentro de aSeparado é um array separando todas as ocorrências

"Gabriel" 
" Fabio" 
" Rodrigo" 
" Isabella" 
" Sandra"
